# gentlemannasa.github.io

Personal cybersecurity portfolio for Samuel Nii Allotey (GentlemanNASA) — built as a living record of hands-on security work, active projects, and continuous learning.

Live at: [gentlemannasa.github.io](https://gentlemannasa.github.io)

---

## What it is

A single-page portfolio designed to go beyond a resume. Instead of listing credentials, it documents real work — offensive security labs, tools built from scratch, civic tech projects, and the learning process behind all of it.

Built for anyone evaluating a cybersecurity candidate who wants to see proof of practice, not just proof of enrollment.

---

## Built with

- Vanilla HTML, CSS, and JavaScript — no frameworks, no dependencies
- Dark cyberpunk aesthetic with a cyan, magenta, and amber palette
- Syne and Outfit fonts via Google Fonts
- Fully responsive — canvas and 3D elements disabled on mobile for performance
- Deployed via GitHub Pages

---
